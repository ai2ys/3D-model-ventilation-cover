# Ventilation Cover

[![CC BY-SA 4.0](https://img.shields.io/badge/License-CC%20BY--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-sa/4.0/)


This is a 3D model of a ventilation cover designed for 3D printing. The cover features a line/grid pattern that allows air to flow through while providing protection.
It is the first draft for a fast PoC.

## Features

- rectangular shape as base
- cylinder for mounting it to tube in the wall
- line/grid pattern for air flow
